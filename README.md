PROJECT TITLE
Sales Data Analysis | PostgreSQL + Power BI

OVERVIEW
This project analysez sales data across multiple store slocations and product categories. 
The data was imported from CSV files into PostgreSQL, consolidated, cleaned and visualized in Power BI to identify trends and financial insight.

TOOLS
- PostgreSQL/pgAdmin
- Power BI
- CSV (raw data)

PROCESS
  1. Loaded 4 CSV files into PostgreSQL tables
  2. Merged data into a single, consolidated table
  3. Handled missing values and duplicated
  4. Created calculated fields (Total Amount, Profit)
  5. Performed descriptive analysis and visualized results in Power BI
 
KEY INSIGHTS
- Sales peak in December and January; lowest in June and August
- Store performance doesn't vary significantly across locations/countries


NOTE
Dataset is used for educational and portofolio purposes.
